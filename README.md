# Dotfiles

A lot of it will only work on OSX.

## Prerequisites

[mxcl/homebrew]
[kennethreitz/osx-gcc-installer]
[sstephenson/rbenv]

## Installation

`git clone git://github.com/jcoene/dotfiles ~/.dotfiles
cd ~/.dotfiles
rake install`

## Setup

- Use zsh, pow and rbenv
- Projects go in ~/Projects
